<p> • พอเราได้ทำการกรอกข้อมูลอัพเดทแล้วก็จะแบ่งออกแบ่ง Box นะครับ </p>

<img src="ref/Monitorbox.png" width="400" height="750">
<p> • พอเรากดเลือก Box ที่เราจะดูข้างในก็จะเป็นการโชว์ข้อมูลที่เราได้ทำการ </p>
<p> กรอกอัพเดทการเจริญเติบโตเอาไว้นะครับนี้ก็จะเป็นตัวอย่างคร่าวๆให้รับชม </p>
<p> ที่จะเพิ่มเข้ามาหน้านี้ก็คือเป็นในส่วนของ Color นะครับก็จะบ่งบอกถึง Level นั้นๆว่า </p>
<p> สีของน้องๆอยู่สีไหนสวยไหม แข็งแรงไหมอะไรประมานนี้นะครับ </p>
<img src="ref/Pandacolor.png" width="400" height="750">

<p> Graph ที่สองจะแสดงสีของน้องๆว่าอยู่ Level ไหนอะไรประมานนี้ครับ </p>
